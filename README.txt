Plugin se stará o správně zobrazování počtu komentářů na stránce webu. Je možné si nastavit vlastní řetězce, které se při určitém počtu budou zobrazovat.

== Description ==

Plugin opravuje chybu komentářů, které nejsou stavěné na české skloňování. V jednoduché administraci můžete nastavit své vlastní řetězce pro určitý počet komentářů.

A few notes about the sections above:


== Installation ==

Postup instalace


1. Stáhněte plugin Česk=-komentáře.zip
2. Přejděte do administrace wordpressu
3. Uložte plugin přes administraci pluginů ve wordpressu
4. Aktivujte plugin České komentáře

Druhá možnost

1. Přejděte do administrace pluginů ve wordpressu
2. Najděte plugin České komentáře
3. Nainstalujte plugin a zaktivujte
